# Phishing Email Detection Project

## Dataset

This project uses the Phishing Email Dataset from Kaggle (link below).  
The dataset contains ~82,500 emails (42,891 phishing, 39,595 legitimate).  

Due to license restrictions, the dataset itself is not included.  
Please download it via Kaggle: https://www.kaggle.com/datasets/naserabdullahalam/phishing-email-dataset
